# Minecraft-Server.eu Voting Bot

A voting bot for Minecraft-Server.eu. Supports multiple servers to vote for and usernames.

## Requirements

- Python 3
- Pip
- Pipenv (`pip install pipenv`)
- Mozilla Firefox
